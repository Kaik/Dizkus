Installation
------------

* Zippaket auspacken, es erscheint folgende Dateistruktur:

  html
  |
  -- includes
  |  |
  |  -- search
  |
  -- modules
     |
     -- Dizkus
        |
        -- weitere Unterverzeichnisse

  Den Inhalt von modules ins Zikula modules Verzeichnis kopieren
  (nicht modules/Modules!!)
  Die Datei backforum.php aus dem Dizkusverzeichnis ins Zikula Verzeichnis
  kopieren

* Modulliste neu erzeugen lassen
* Dizkus installieren und aktivieren
* Hooks nach Bedarf für Dizkus einstellen.

Weitere Informationen zu den einzelnen Features gibts in

mail2forum.txt
rss2forum.txt
Dizkuscomments.txt

Information zum Berechtigungssystem gibts in

upgrade.txt
